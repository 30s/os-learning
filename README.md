#Linux内核学习
=====

本系列是我学习Linux的笔记。关于内核实现的一些原理的具体内容，《Linux内核设计与实现》（Linux Kernel Development，简称LKD）已经比较全面了，网络上的资料也非常多。这系列博文主要想站在初学者的角度，在了解How之前，先了解What跟Why，从而对内核有个稳固的大局观。同时会寻根究底的方式，找到对应的内核代码，满足一下个人的考究爱好。
